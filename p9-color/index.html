Color

In CSS, the color property is used to specify the text color of an element. You can set the color of text using various methods, such as named colors, hexadecimal values, RGB values, or HSL values.

Here are some examples:

Named Colors

CSS provides a set of predefined color names that you can use directly.

Hexadecimal Values

You can also use hexadecimal values to represent colors.

RGB Values

RGB (Red, Green, Blue) values allow you to specify a color using the intensity of each color channel.

HSL Values

HSL (Hue, Saturation, Lightness) values provide another way to represent colors.
