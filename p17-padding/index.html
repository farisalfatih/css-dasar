Padding

Padding dalam CSS adalah ruang tambahan yang diberikan di sekitar konten dalam suatu elemen HTML. Padding berfungsi untuk memberikan jarak antara konten elemen dan batas (border) elemen tersebut. Dengan kata lain, padding menentukan seberapa jauh konten elemen berada dari tepi elemen tersebut.

Anda dapat mengatur padding menggunakan properti CSS padding dan memberikan nilai yang sesuai. Properti ini dapat memiliki satu hingga empat nilai, yang mewakili padding untuk sisi atas, sisi kanan, sisi bawah, dan sisi kiri elemen, secara berurutan. Berikut adalah contoh penggunaan:

/* Satu nilai: padding untuk semua sisi */
padding: 10px;

/* Dua nilai: padding untuk atas/bawah dan kanan/kiri */
padding: 10px 20px;

/* Tiga nilai: padding untuk atas, kanan/kiri, dan bawah */
padding: 10px 20px 30px;

/* Empat nilai: padding untuk atas, kanan, bawah, dan kiri */
padding: 10px 20px 30px 40px;

Dalam contoh di atas, 10px adalah padding atas, 20px adalah padding kanan, 30px adalah padding bawah, dan 40px adalah padding kiri.

Penting untuk dicatat bahwa padding menambahkan ruang di sekitar konten tanpa mengubah ukuran elemen itu sendiri. Properti ini berguna untuk memberikan ruang di sekitar elemen, sehingga tata letak halaman web dapat diatur dengan lebih baik.
