CSS Text

Pada pembelajaran css text kali ini saya akan merangkum pelajaran yang saya dapat dari pak pukis mulai tari pertemuan ke 13 sampai 14 yang membahas tentang text. Dari pembelajaran berikut saya mendapatkan beberapa tag antara lain :

  1. font-family

    2. Menentukan jenis font yang akan digunakan untuk suatu elemen. Misalnya, font-family: 'Arial', sans-serif; akan menggunakan font Arial jika tersedia, jika tidak, akan mencari font sans-serif sebagai alternatif.

  2. font-size

    3. Menentukan ukuran font. Misalnya, font-size: 16px; akan mengatur ukuran font menjadi 16 piksel.

  3. font-weight

    4. Mengatur ketebalan font. Nilai dapat berupa angka (100 hingga 900), 'normal', 'bold', atau 'bolder'.

  4. font-style

    5. Menentukan gaya font seperti italic atau normal. Misalnya, font-style: italic;.

  5. text-decoraton

    6. Menentukan dekorasi teks seperti garis bawah atau garis melintang. Misalnya, text-decoration: underline;.

  6. text-transform

    7. Mengubah gaya huruf teks, seperti membuat teks menjadi huruf kapital. Misalnya, text-transform: uppercase;.

  7. text-align

    8. Menentukan penataan teks dalam suatu elemen, seperti 'left', 'right', 'center', atau 'justify'.

  8. color

    9. Menentukan warna teks. Misalnya, color: #333; akan memberikan warna teks abu-abu gelap.

  9. text-shadow

    10. Menambahkan bayangan pada teks dengan memberikan nilai offset x, offset y, blur radius, dan warna. Misalnya, text-shadow: 2px 2px 4px #888888;.

  10. word-spacing

    11. Menentukan jarak antara kata-kata dalam teks. Misalnya, word-spacing: 2px; akan memberikan jarak 2 piksel antara setiap kata.

  11. latter-spacing

    12. Menentukan jarak antara huruf dalam teks. Misalnya, letter-spacing: 1px; akan memberikan jarak 1 piksel antara setiap huruf.

  12. :hover

    13. Pseudo-class yang memungkinkan Anda menentukan gaya saat elemen tersebut dihover oleh pengguna. Misalnya, a:hover { color: red; } akan mengubah warna teks menjadi merah saat link dihover.

  13. :active

    14. Pseudo-class yang memungkinkan Anda menentukan gaya saat elemen sedang dalam keadaan aktif (biasanya saat tombol mouse ditekan).

  14. :visited

    15. Pseudo-class yang memungkinkan Anda menentukan gaya untuk link yang telah dikunjungi oleh pengguna. Sebagai contoh, a:visited { color: purple; } akan memberikan warna ungu pada link yang telah dikunjungi.
