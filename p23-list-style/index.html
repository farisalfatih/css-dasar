List Style

Dalam CSS, properti list-style digunakan untuk mengontrol gaya tampilan daftar (ul, ol, dan dl). Properti ini memungkinkan Anda untuk mengatur tiga aspek utama dari tampilan daftar: jenis marker, jenis bilangan (jika menggunakan daftar terurut), dan gambar latar belakang untuk daftar.

Nilai utama dari properti list-style adalah kombinasi dari tiga properti terpisah: list-style-type, list-style-position, dan list-style-image. Di bawah ini adalah penjelasan singkat tentang masing-masing properti:

  1. list-style-type: Menentukan jenis marker atau bilangan pada daftar. Beberapa nilai umumnya melibatkan:
    • none: Tidak ada marker atau bilangan.
    • disc: Marker berbentuk bulatan (default untuk daftar tak terurut).
    • decimal: Bilangan angka (default untuk daftar terurut).
    • circle, square, dll.: Jenis marker khusus.
  2. list-style-position: Menentukan posisi marker atau bilangan. Beberapa nilai umumnya melibatkan:
    • outside: Marker atau bilangan berada di luar konten daftar.
    • inside: Marker atau bilangan berada di dalam konten daftar.
  3. list-style-image: Menentukan gambar yang digunakan sebagai marker pada daftar. Nilai umumnya adalah URL gambar.

Anda dapat menggabungkan ketiga properti ini dalam satu aturan CSS menggunakan properti list-style. Misalnya:

/* Kombinasi jenis, posisi, dan gambar marker */
list-style: square inside url('custom-marker.png');
