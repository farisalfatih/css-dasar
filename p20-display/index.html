Display

display adalah properti CSS yang digunakan untuk mengontrol bagaimana suatu elemen HTML akan ditampilkan di halaman. Properti ini memungkinkan Anda mengubah perilaku default dari elemen, menentukan apakah elemen tersebut akan dirender sebagai blok, inline, atau tata letak fleksibel (flexbox), grid, dan sebagainya.

Beberapa nilai umum untuk properti display antara lain:

block

block: Membuat elemen menjadi blok, menempatkannya di baris baru dan memanjanginya sepanjang lebar dari kontainer.

div {display: block;}

inline

inline: Membuat elemen menjadi inline, menempatkannya di dalam konten dan hanya memanjangi lebar yang diperlukan oleh kontennya.

div {display: inline;}

inline-block

inline-block: Kombinasi dari inline dan block. Elemen ini tetap di dalam konten, namun memungkinkan penyesuaian ukuran dan propertinya seperti elemen blok.

div {display: inline-block;}

none

none: Menyembunyikan elemen dari tampilan halaman. Elemen ini tidak akan terlihat dan tidak memengaruhi tata letak halaman.

div {display: none;}

flex

flex: Membuat elemen menjadi kontainer fleksibel, memungkinkan penataan elemen anak secara dinamis dalam satu atau dua dimensi.

div {display: flex;}

grid

grid: Membuat elemen menjadi kontainer grid, memungkinkan penataan elemen anak dalam grid dua dimensi.

div {display: grid;}

Nilai dari properti display ini memberikan kontrol yang sangat besar terhadap tata letak elemen dalam desain halaman web, dan pemahaman yang baik tentang bagaimana menggunakan nilai-nilai ini dapat memudahkan dalam mencapai struktur tata letak yang diinginkan.
