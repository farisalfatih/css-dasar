CSS Selector

Universal Selector: Use the * sign to select all elements in the html that are comprehensive

Element Selector: Selects HTML elements based on their tag name.

Class Selector: Selects elements with a specific class attribute.

ID Selector: Selects a specific element with a unique ID attribute.

Child Selector: Selects an element that is a direct child of another specified element.

Descendant Selector: Selects an element that is a descendant of another specified element.
