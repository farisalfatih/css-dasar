Table Styles

Untuk membuat tabel lebih cantik dan estetis, Anda dapat menggunakan berbagai properti CSS untuk mengatur warna, font, dan tata letak. Berikut adalah beberapa properti CSS yang dapat Anda gunakan untuk meningkatkan tampilan tabel:

Name Property Explain Exampel Code
Background Color dan Text Color: Mengatur warna latar belakang dan warna teks pada sel tabel. background-color: #fff;
color: #333;
Padding dan Margin: Menetapkan jarak dalam (padding) dan luar (margin) pada sel-sel dan elemen tabel. padding: 10px;
margin: 5px;
border: Menentukan gaya, warna, dan lebar batas pada sel dan elemen tabel. border: 1px solid #ddd;
Hover Effect: Menambahkan efek saat mouse diarahkan pada baris tabel. tr:hover{background-color: #f5f5f5;}
Font Style: Menetapkan gaya huruf, ukuran, dan jenis font. font-family: 'Arial', sans-serif;
font-size: 14px;
font-weight: normal;
Rounded Corners: Memberikan sudut yang lebih lembut pada sel dan elemen tabel. border-radius: 5px;
Alternating Row Colors: Memberikan warna yang berbeda untuk baris ganjil dan genap. tr:nth-child(even){ background-color: #f9f9f9; }
Box Shadow: Menambahkan bayangan untuk memberikan efek visual kedalaman. box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
Transitions: Menambahkan efek transisi halus untuk perubahan warna atau properti lainnya. transition: background-color 0.3s ease;
